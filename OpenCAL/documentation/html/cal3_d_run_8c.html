Creates an object of type calRunDef3D, sets its records and returns it as a pointer; it defines the cellular automaton simulation structure.

Parameters
ca3DPointer to the cellular automaton structure.
initial_stepInitial simulation step; default value is 0.
final_stepFinale step; if it is 0, a loop is obtained. In order to set final_step to 0, the constant CAL_RUN_LOOP can be used.
UPDATE_MODEUpdate mode: explicit on or explicit off (implicit).
